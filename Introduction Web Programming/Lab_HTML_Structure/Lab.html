style : การใช้ CSS แบบ Inline

style : การใช้ CSS แบบ Inline

img src , alt , width , height src : ระบุที่อยู่ของไฟล์
alt : เหมือนเป็นการระบุเพิ่มเติมของรูปภาพ
width : กำหนดความกว้าง
height : กำหนดความสูง

ul type ระบุประเภท

start : ตั้งค่าตัวเริ่มเต้น
type : ระบุประเภท
table align , bgcolor , width align : จัดตำแหน่งบนซ้ายกลางขวา
bgcolor : การปรับสีของตาราง

Differentiation Tag

Tag ความเหมือน ความแตกต่าง
hr & br ขึ้นบรรทัดใหม่เหมือนกัน hr จะเป็นเส้นขีด br จะไม่มีเส้นขีด
ul & ol แสดงข้อมูลแบบลิสต์ ul เป็นการลิสต์แบบไม่มีลำดับ โดยใช้เครื่องหมาย Bullet list กำกับ
ol จะเป็นการลิสต์แบบเป็นลำดับ โดยใช้ตัวเลขกำกับ
select & checkbox & radio button เป็นการเลือกเหมือนกัน select จะเป็นการเลือกข้อมูลแบบ Dropdown ส่วน checkbox เป็นการเลือกข้อมูลโดยการติ๊กถูก และ radio button เป็นการเลือกข้อมูลโดยการคลิกที่วงกลมและเป็นการเลือกอย่างใดอย่างนึง
div & span จัดกลุ่มข้อมูล div จะมี display : block เป็นค่า Default ทำให้เราสร้าง div หลายๆอันจะอยู่ด้านล่างเรียงลงไปแต่ span จะเป็นแบบ inline ทำให้จะอยู่บรรทัดเดียวกัน
script & style เป็นการดึงภาษาอื่นๆเข้ามาใช้ script เป็นการดึง Javascipt มาใช้ ส่วน style เป็นการดึง CSS เข้ามาใช้
